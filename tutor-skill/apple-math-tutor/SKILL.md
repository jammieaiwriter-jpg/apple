# 🍎 Apple 數學助教技能 (Apple Math Tutor)

此技能將「錯題分析」與「教材更新」流程自動化，確保 Apple 的練習內容能與她的學習進度緊密結合。

## 🛠 工作流程

當用戶上傳新的錯題截圖、提到有新的練習結果、或想分析遊戲錯題記錄時，請執行以下步驟：

### 1. 雙管道錯題掃描

**管道 A：截圖錯題**
- 檢查 `/Users/emma/Desktop/apple/錯題` 資料夾中最新的圖片（比對 questions.json 裡已有的檔名，找出尚未處理的新圖）。
- 辨識題目內容、正確答案，並撰寫一段語氣親切、包含解題關鍵點的「💡 提示」(Hint)。
- 將新題目資訊同步寫入 `/Users/emma/Desktop/apple/questions.json`。

**管道 B：遊戲錯題（game-mistakes.json）**
- 檢查 `/Users/emma/Desktop/apple/game-mistakes.json` 是否存在，若存在則讀取。
- 此檔案記錄 Apple 在 index.html 遊戲中答錯的題型（unit ID、時間、是否用過提示）。
- 將遊戲錯誤統計納入課綱弱點分析，但**不需要**寫入 questions.json（截圖才寫入）。
- 分析完成後，可清空此檔案（設為空陣列 `[]`）以避免重複處理。

### 2. 課綱對照與進度紀錄

依照幼兒園大班(L)(下冊)數學檢定課綱，將題目歸類至以下 **6 大能力群組**（不需背 88 個回別，直接對應遊戲模式）：

| 能力群組 | 涵蓋的課綱題型（舉例） | 對應遊戲模式 |
| :--- | :--- | :--- |
| **💰 錢幣計算** | 硬幣組合、購買物品加減、複合幣值、路程距離金額計算 | `coin` |
| **🐸 數序規律** | 數序順逆數(2~10)、等差跳數、圖形數量計數、倍數跳數 | `jump` |
| **🧠 加減逆算** | 加減計算、逆向填空、三步驟計算、文選加減、天平重量、加減符號判斷 | `reverse` |
| **🍬 故事算式理解** | 算式轉故事、故事轉算式、加減語意辨識、未知數故事理解 | `story` |
| **⏰ 時間認知** | 整點、半點、四分點辨識與連線 | `clock` |
| **🔢 奇偶數** | 奇偶數判別、尾數辨識 | `even` |
| **📦 空間圖形** | 立體/方塊圖形辨認與補合、立體積木總數、圖層判斷、左右上下前後方位、圖形移動旋轉、展開圖 | `space` |

歸類時，**不需要指出確切的「回別N」**，只要正確對應能力群組即可，例如：「積木補合 → 空間圖形 (space)」。若題目橫跨多個群組（如倍數應用文選），選最主要的技能。

### 3. 更新 apple-mistakes.md

- 在「任務與精通度追蹤表」的「備註」欄位加入新的弱點觀察（包含課綱回別）。
- 在「錯題詳細記錄」中增加該次錯誤的分析與輔導建議。
- 日期欄位更新為今日日期。

### 4. 生成針對性練習

- 修改 `/Users/emma/Desktop/apple/index.html` 中的 `generateQuestion()` 邏輯。
- 根據本次錯題的類型，增加或加強該單元的隨機生成邏輯，例如：
  - Apple 圖形補合題答錯 → 在 `space` 模式加強補合類視覺題
  - Apple 數序答錯 → 在 `jump` 模式加入她常錯的跳數步驟
  - Apple 加減逆算答錯 → 調整 `reverse` 的數字範圍
- **目標**：讓 Apple 在玩網頁遊戲時，有更高機率遇到她剛錯過的觀念。

### 5. 更新遊戲錯題機制（若 index.html 遊戲尚未支援 game-mistakes.json）

確認 index.html 裡的 `checkAnswer()` 函式中的答錯記錄，除了寫入 localStorage 之外，也有在每次 `downloadReport()` 時提示用戶：可以把 `report.md` 放回資料夾讓 Claude 分析。

## 📋 遊戲模式與課綱對應速查

| 遊戲模式 ID | 遊戲名稱 | 主要涵蓋課綱範圍 |
| :--- | :--- | :--- |
| `coin` | 💰 金幣小管家 | 硬幣組合、硬幣加減、購買物品計算 |
| `jump` | 🐸 高階跳跳蛙 | 數序順逆數（2~10）、圖形計數、倍數跳數 |
| `reverse` | 🧠 逆向大偵探 | 加減逆算、三步驟計算、文選加減、天平計算 |
| `story` | 🍬 故事算式小編劇 | 算式與生活故事互相翻譯、理解加減語意 |
| `clock` | ⏰ 時間小偵探 | 整點、半點、四分點認識與連線 |
| `even` | 🔢 雙數小捕手 | 奇偶數判別 |
| `space` | 📦 空間小超人 | 立體補合、立體積木總數、方位辨識、圖層判斷、圖形移動 |
