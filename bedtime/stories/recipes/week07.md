# 故事配方卡 — week07（表達）

- 切點（focus，固定）：從身體感覺認出自己的心情
- 故事形狀：等待型（安靜守候一件事慢慢發生）
- 主角：一隻想念朋友的小狐狸
- 場景：窗邊的月光花園
- 情緒曲線：緊張 → 鼓起勇氣 → 暖暖的
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：唱歌
- 陪伴角色：一隻打瞌睡的小貓
- 主導感官：視覺
- 睡前收束：動作變慢、被子變暖、世界變安靜
- 結尾句式：身體放鬆式  → 範式「雨芯，晚安。讓～慢慢～，～地睡。」
- 禁用元素：
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「森林邊的小木屋」
  - 不要和前兩晚用同一種解決方式或結尾句式
