# 故事配方卡 — week12-3（成長）

- 切點（focus，固定）：以前需要幫忙的事，現在能自己完成
- 故事形狀：來訪型（到別人的地方作客、靠對話推進）
- 主角：一滴小水珠
- 場景：閣樓上的舊鐘塔
- 情緒曲線：緊張 → 驚喜 → 安穩
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：分享
- 陪伴角色：一把溫柔的小掃帚
- 主導感官：觸覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：明日式  → 範式「雨芯，晚安。～還在那裡等明天，先安心睡吧。」
- 禁用元素：
  - 不要循線索找一個遺失的東西或聲音
  - 不要重複場景「夜晚的麵包店」
  - 不要主角慢慢從一個狀態變成另一個
  - 不要重複場景「書頁之間的微小世界」
  - 不要和前兩晚用同一種解決方式或結尾句式
