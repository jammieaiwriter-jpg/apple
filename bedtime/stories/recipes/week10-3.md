# 故事配方卡 — week10-3（選擇）

- 切點（focus，固定）：聽聽會受到影響的人的想法
- 故事形狀：來訪型（到別人的地方作客、靠對話推進）
- 主角：一台舊舊的小收音機
- 場景：池塘邊的荷葉舞台
- 情緒曲線：孤單 → 猶豫 → 放鬆
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：擁抱
- 陪伴角色：窗外的月亮
- 主導感官：視覺
- 睡前收束：顏色變淡、風變輕、念頭一個一個睡著
- 結尾句式：明日式  → 範式「雨芯，晚安。～還在那裡等明天，先安心睡吧。」
- 禁用元素：
  - 不要一起動手做或修一個東西
  - 不要重複場景「書頁之間的微小世界」
  - 不要循線索找一個遺失的東西或聲音
  - 不要重複場景「月亮背面的小郵局」
  - 不要和前兩晚用同一種解決方式或結尾句式
