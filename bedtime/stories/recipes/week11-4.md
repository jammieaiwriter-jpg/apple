# 故事配方卡 — week11-4（想像）

- 切點（focus，固定）：不只照著別人畫，畫出自己的答案
- 故事形狀：等待型（安靜守候一件事慢慢發生）
- 主角：一朵棉花雲
- 場景：夜市尾端的小燈籠攤
- 情緒曲線：擔心 → 鼓起勇氣 → 安穩
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：傾聽
- 陪伴角色：一條舊圍巾
- 主導感官：味覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：比喻式  → 範式「雨芯，晚安。今晚就像～，～。」
- 禁用元素：
  - 不要出門遠行、一路過關
  - 不要重複場景「抽屜裡的玩具城」
  - 不要一起動手做或修一個東西
  - 不要重複場景「圖書館深處的安靜角落」
  - 不要和前兩晚用同一種解決方式或結尾句式
