# 故事配方卡 — week09-3（耐心）

- 切點（focus，固定）：留心還不明顯的小變化
- 故事形狀：相遇型（兩人各做各的後相遇、化解小誤會）
- 主角：一台舊舊的小收音機
- 場景：海邊的小燈塔
- 情緒曲線：孤單 → 猶豫 → 安心
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：合作
- 陪伴角色：一隻不會說話的紙鶴
- 主導感官：聽覺
- 睡前收束：動作變慢、被子變暖、世界變安靜
- 結尾句式：比喻式  → 範式「雨芯，晚安。今晚就像～，～。」
- 禁用元素：
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「書頁之間的微小世界」
  - 不要主角慢慢從一個狀態變成另一個
  - 不要重複場景「夜市尾端的小燈籠攤」
  - 不要和前兩晚用同一種解決方式或結尾句式
