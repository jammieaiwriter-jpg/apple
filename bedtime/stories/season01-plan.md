# 第一季睡前故事計畫

## 內容節奏：主題 → 切點 → 故事

- 一週一個核心詞主題；每個主題拆成 **4–5 個「切點」(facet)**，每個切點是一個明確的教學角度，由角色行動演出（不說教）。
- **一切點 = 一篇故事**（1:1）。切點有順序，組成一個溫和的進程；夜間頁在本週主題內每晚自動換一篇 `adult_verified` 故事，照切點順序輪播，但每篇仍可單獨聽懂。
- 孩子一週聽五晚，4–5 個切點剛好填滿一週、幾乎不重複；切點數可依主題微調。
- 命名：`weekNN`（切點一／主打）、`weekNN-2`、`weekNN-3`…。新增故事預設 `pending_adult_review`，成人試聽通過才進夜間輪播。
- 資料：`catalog.json` 的每週 episode 用 `facets` 宣告切點清單，每篇故事用 `focus` 標記它對應哪個切點。
- 切點 3「分辨善意與不友善」對六歲孩子用「認出讓你不舒服的對待、可以不忍耐、可以找別的朋友」的講法，不給角色貼「壞」標籤、不勉強和好。

## 共通規格

- 共用森林世界，但每集獨立可懂；主角依衝突選擇，不固定輪替。
- 維持 `schema_version: 2`、六個 `sections` 與 `wind_down`。
- 核心詞由角色行動呈現；週末問題問角色做了什麼，不考名詞定義。
- 老山羊長故事嵌入既有段落，不新增第七段；成語僅為可選延伸。
