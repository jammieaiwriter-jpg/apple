# 故事配方卡 — week09-4（耐心）

- 切點（focus，固定）：著急時先安頓自己的身體
- 故事形狀：旅程型（出門遠行、一路過關）
- 主角：一片小小的雪花
- 場景：會發光的蘑菇山洞
- 情緒曲線：興奮 → 鼓起勇氣 → 安穩
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：傾聽
- 陪伴角色：一隻安靜的螢火蟲
- 主導感官：嗅覺
- 睡前收束：動作變慢、被子變暖、世界變安靜
- 結尾句式：明日式  → 範式「雨芯，晚安。～還在那裡等明天，先安心睡吧。」
- 禁用元素：
  - 不要主角慢慢從一個狀態變成另一個
  - 不要重複場景「夜市尾端的小燈籠攤」
  - 不要兩人各做各的後相遇、化解小誤會
  - 不要重複場景「海邊的小燈塔」
  - 不要和前兩晚用同一種解決方式或結尾句式
