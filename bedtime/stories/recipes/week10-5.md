# 故事配方卡 — week10-5（選擇）

- 切點（focus，固定）：得到新資訊後，可以重新調整選擇
- 故事形狀：等待型（安靜守候一件事慢慢發生）
- 主角：一團毛線球
- 場景：雲上的小陽台
- 情緒曲線：緊張 → 被理解 → 安穩
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：唱歌
- 陪伴角色：一隻不會說話的紙鶴
- 主導感官：嗅覺
- 睡前收束：動作變慢、被子變暖、世界變安靜
- 結尾句式：比喻式  → 範式「雨芯，晚安。今晚就像～，～。」
- 禁用元素：
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「池塘邊的荷葉舞台」
  - 不要兩人各做各的後相遇、化解小誤會
  - 不要重複場景「下雪的屋頂」
  - 不要和前兩晚用同一種解決方式或結尾句式
