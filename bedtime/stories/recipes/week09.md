# 故事配方卡 — week09（耐心）

- 切點（focus，固定）：等待時不急著干擾結果
- 故事形狀：來訪型（到別人的地方作客、靠對話推進）
- 主角：一隻小章魚
- 場景：書頁之間的微小世界
- 情緒曲線：好奇 → 猶豫 → 暖暖的
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：想像
- 陪伴角色：一把溫柔的小掃帚
- 主導感官：觸覺
- 睡前收束：呼吸變慢、身體變暖、眼睛變重
- 結尾句式：物件呼應式  → 範式「雨芯，晚安。～（故事裡的物件）也～了，妳也～。」
- 禁用元素：
  - 不要輕輕守護、照顧一個更小更弱的對象
  - 不要重複場景「圖書館深處的安靜角落」
  - 不要和前兩晚用同一種解決方式或結尾句式
