# 故事配方卡 — week07-4（表達）

- 切點（focus，固定）：具體說出需要什麼幫助
- 故事形狀：變化型（主角慢慢從一個狀態變成另一個）
- 主角：一隻夜裡才醒的小貓頭鷹
- 場景：雲上的小陽台
- 情緒曲線：擔心 → 發現 → 暖暖的
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：放下
- 陪伴角色：一隻不會說話的紙鶴
- 主導感官：味覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：明日式  → 範式「雨芯，晚安。～還在那裡等明天，先安心睡吧。」
- 禁用元素：
  - 不要兩人各做各的後相遇、化解小誤會
  - 不要重複場景「抽屜裡的玩具城」
  - 不要一起動手做或修一個東西
  - 不要重複場景「書頁之間的微小世界」
  - 不要和前兩晚用同一種解決方式或結尾句式
