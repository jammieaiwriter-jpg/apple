# 故事配方卡 — week05（傾聽）

- 切點（focus，固定）：停下自己的聲音，讓身體安靜
- 故事形狀：來訪型（到別人的地方作客、靠對話推進）
- 主角：一隻紙摺的小鶴
- 場景：閣樓上的舊鐘塔
- 情緒曲線：興奮 → 慢慢明白 → 安心
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：觀察
- 陪伴角色：一隻不會說話的紙鶴
- 主導感官：觸覺
- 睡前收束：香味變淡、燈光變柔、聲音變小
- 結尾句式：物件呼應式  → 範式「雨芯，晚安。～（故事裡的物件）也～了，妳也～。」
- 禁用元素：
  - 不要一起動手做或修一個東西
  - 不要重複場景「結霜的玻璃窗內側」
  - 不要和前兩晚用同一種解決方式或結尾句式
