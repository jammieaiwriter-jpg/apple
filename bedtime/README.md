# 🌙 睡前故事 — Apple 的國語文科目

Apple 學習系統的「國語文」科目，以睡前說故事培養聽、跟讀、思考、用詞。
純靜態 HTML，掛在現有 apple repo 的 GitHub Pages 下：
`https://jammieaiwriter-jpg.github.io/apple/bedtime/`

> 設計必須遵守 [`../LEARNING_DNA.md`](../LEARNING_DNA.md) 的靈魂方針，特別是：
> - **自尊保護**：不「教導」、不「考試」。二選一與「為什麼」要包裝成她替主角做決定、提供「秘密情報」，
>   讓答案像是「她自己想出來的」，不是被測驗。
> - **善用記憶優勢**：她背誦記憶強 → 用目標詞建詞彙庫與故事聯想（隔天故事自然重現前一晚的詞）。
> - **動態適應**：避免重複已會的內容讓她無聊。

## 這是什麼
- 每晚一幕故事，聊天泡泡逐句顯示、`speechSynthesis` 朗讀；完成後下次自動前往新的一幕。
- 每幕埋一個目標詞；幕末讓她替主角二選一（A/B），選擇存進 `localStorage`。
- 隔天故事旁白「記得她昨天的選擇」（偽分支：固定主軸 + 旁白回應），建立期待感。
- 週末用五幕縮圖讓她重述整週故事，家長旁邊勾「需提示 / 可獨立」。

## 不做（首版砍掉）
錄音、答題式詞彙熟練度追蹤、動畫、真分支（2^5 種結局維護不動）。

## 內容來源
故事 JSON 由 AI **預先生成**（媽媽無法自己寫故事），家長只選本週主題
（友情 / 自然 / 情緒 / 生活）。每週補一個 `stories/weekNN.json`，並由成人審閱後標記
`"review_status": "adult_verified"` 才能上線。

## 技術手法（沿用 apple 既有做法）
- 朗讀：`speechSynthesis`，`lang: 'zh-TW'`、`rate: 0.9`。
- 故事資料：JSON 與顯示分離（概念同 apple `questions.json` 的題庫分離）。
- 進度：`localStorage`。

詳細定案與建造順序見 [PLAN.md](PLAN.md)。
