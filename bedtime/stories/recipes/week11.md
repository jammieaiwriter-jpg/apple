# 故事配方卡 — week11（想像）

- 切點（focus，固定）：從平常事物看見不同可能
- 故事形狀：來訪型（到別人的地方作客、靠對話推進）
- 主角：一隻夜裡才醒的小貓頭鷹
- 場景：森林邊的小木屋
- 情緒曲線：害羞 → 發現 → 暖暖的
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：觀察
- 陪伴角色：一隻不會說話的紙鶴
- 主導感官：聽覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：願式  → 範式「雨芯，晚安。願～，做一個～的夢。」
- 禁用元素：
  - 不要安靜守候一件事慢慢發生
  - 不要重複場景「雲上的小陽台」
  - 不要和前兩晚用同一種解決方式或結尾句式
