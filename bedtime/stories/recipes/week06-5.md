# 故事配方卡 — week06-5（嘗試）

- 切點（focus，固定）：休息、調整，再決定下一步
- 故事形狀：來訪型（到別人的地方作客、靠對話推進）
- 主角：一塊害羞的小餅乾
- 場景：森林邊的小木屋
- 情緒曲線：擔心 → 鼓起勇氣 → 安心
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：分享
- 陪伴角色：一隻安靜的螢火蟲
- 主導感官：觸覺
- 睡前收束：呼吸變慢、身體變暖、眼睛變重
- 結尾句式：願式  → 範式「雨芯，晚安。願～，做一個～的夢。」
- 禁用元素：
  - 不要輕輕守護、照顧一個更小更弱的對象
  - 不要重複場景「圖書館深處的安靜角落」
  - 不要兩人各做各的後相遇、化解小誤會
  - 不要重複場景「一格一格的棋盤花園」
  - 不要和前兩晚用同一種解決方式或結尾句式
