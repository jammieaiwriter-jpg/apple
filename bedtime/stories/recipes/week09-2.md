# 故事配方卡 — week09-2（耐心）

- 切點（focus，固定）：在等待中持續做能做的照顧
- 故事形狀：變化型（主角慢慢從一個狀態變成另一個）
- 主角：一顆等待發芽的種子
- 場景：夜市尾端的小燈籠攤
- 情緒曲線：興奮 → 猶豫 → 暖暖的
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：擁抱
- 陪伴角色：一條舊圍巾
- 主導感官：視覺
- 睡前收束：呼吸變慢、身體變暖、眼睛變重
- 結尾句式：願式  → 範式「雨芯，晚安。願～，做一個～的夢。」
- 禁用元素：
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「書頁之間的微小世界」
  - 不要和前兩晚用同一種解決方式或結尾句式
