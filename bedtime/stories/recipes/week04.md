# 故事配方卡 — week04（合作）

- 切點（focus，固定）：先找到大家共同的目標
- 故事形狀：變化型（主角慢慢從一個狀態變成另一個）
- 主角：一隻穿雨鞋的小青蛙
- 場景：圖書館深處的安靜角落
- 情緒曲線：緊張 → 小混亂 → 暖暖的
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：深呼吸
- 陪伴角色：一隻打瞌睡的小貓
- 主導感官：視覺
- 睡前收束：動作變慢、被子變暖、世界變安靜
- 結尾句式：物件呼應式  → 範式「雨芯，晚安。～（故事裡的物件）也～了，妳也～。」
- 禁用元素：
  - 不要出門遠行、一路過關
  - 不要重複場景「雲上的小陽台」
  - 不要和前兩晚用同一種解決方式或結尾句式
