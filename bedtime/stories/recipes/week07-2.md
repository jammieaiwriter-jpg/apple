# 故事配方卡 — week07-2（表達）

- 切點（focus，固定）：用清楚的話說出感受
- 故事形狀：相遇型（兩人各做各的後相遇、化解小誤會）
- 主角：一盞小檯燈精靈
- 場景：抽屜裡的玩具城
- 情緒曲線：害羞 → 小混亂 → 暖暖的
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：等待
- 陪伴角色：一個會回聲的空罐子
- 主導感官：聽覺
- 睡前收束：香味變淡、燈光變柔、聲音變小
- 結尾句式：願式  → 範式「雨芯，晚安。願～，做一個～的夢。」
- 禁用元素：
  - 不要安靜守候一件事慢慢發生
  - 不要重複場景「窗邊的月光花園」
  - 不要和前兩晚用同一種解決方式或結尾句式
