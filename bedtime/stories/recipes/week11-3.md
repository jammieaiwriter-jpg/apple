# 故事配方卡 — week11-3（想像）

- 切點（focus，固定）：把兩個原本不同的東西組合起來
- 故事形狀：製作型（一起動手做或修一個東西）
- 主角：一隻穿雨鞋的小青蛙
- 場景：圖書館深處的安靜角落
- 情緒曲線：擔心 → 被理解 → 暖暖的
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：修補
- 陪伴角色：一隻不會說話的紙鶴
- 主導感官：嗅覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：身體放鬆式  → 範式「雨芯，晚安。讓～慢慢～，～地睡。」
- 禁用元素：
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「森林邊的小木屋」
  - 不要出門遠行、一路過關
  - 不要重複場景「抽屜裡的玩具城」
  - 不要和前兩晚用同一種解決方式或結尾句式
