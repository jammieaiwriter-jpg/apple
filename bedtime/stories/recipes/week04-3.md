# 故事配方卡 — week04-3（合作）

- 切點（focus，固定）：做不順時互相說明與調整
- 故事形狀：來訪型（到別人的地方作客、靠對話推進）
- 主角：一把會唱歌的小雨傘
- 場景：夜晚的麵包店
- 情緒曲線：想念 → 發現 → 滿足
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：唱歌
- 陪伴角色：會打噴嚏的麵粉罐
- 主導感官：嗅覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：身體放鬆式  → 範式「雨芯，晚安。讓～慢慢～，～地睡。」
- 禁用元素：
  - 不要主角慢慢從一個狀態變成另一個
  - 不要重複場景「圖書館深處的安靜角落」
  - 不要兩人各做各的後相遇、化解小誤會
  - 不要重複場景「屋頂水塔旁的小天台」
  - 不要和前兩晚用同一種解決方式或結尾句式
