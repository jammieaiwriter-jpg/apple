# 故事配方卡 — together（一起玩）

- 切點（focus，固定）：比起搶第一，一起玩更開心
- 故事形狀：旅程型（出門遠行、一路過關）
- 主角：一台亮紅色的小賽車
- 場景：黃昏的玩具賽道
- 情緒曲線：興奮 → 衝到第一後的孤單 → 暖暖的
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：深呼吸
- 陪伴角色：一台邊開邊看風景的小巴士
- 主導感官：觸覺
- 睡前收束：速度變慢、燈光變柔、世界變安靜
- 結尾句式：物件呼應式  → 範式「雨芯，晚安。～（故事裡的物件）也～了，妳也～。」
- 禁用元素：
  - 不要讓主角被責罵或處罰才改變（要他自己發現「衝第一卻只剩自己」）
  - 不要把主角設成可被認出的小女孩或腳踏車（避免孩子對號入座、覺得被指責）
  - 不要說教收尾；道理只放 prologue
