# 故事配方卡 — week07-3（表達）

- 切點（focus，固定）：不舒服時說「不要、請停下來」
- 故事形狀：製作型（一起動手做或修一個東西）
- 主角：一支愛畫畫的蠟筆
- 場景：書頁之間的微小世界
- 情緒曲線：想念 → 慢慢明白 → 放鬆
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：修補
- 陪伴角色：一隻安靜的螢火蟲
- 主導感官：觸覺
- 睡前收束：動作變慢、被子變暖、世界變安靜
- 結尾句式：物件呼應式  → 範式「雨芯，晚安。～（故事裡的物件）也～了，妳也～。」
- 禁用元素：
  - 不要安靜守候一件事慢慢發生
  - 不要重複場景「窗邊的月光花園」
  - 不要兩人各做各的後相遇、化解小誤會
  - 不要重複場景「抽屜裡的玩具城」
  - 不要和前兩晚用同一種解決方式或結尾句式
