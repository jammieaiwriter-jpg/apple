# 故事配方卡 — week05-3（傾聽）

- 切點（focus，固定）：不確定時先詢問，不急著猜答案
- 故事形狀：製作型（一起動手做或修一個東西）
- 主角：一盞小檯燈精靈
- 場景：雨後的彩虹橋下
- 情緒曲線：孤單 → 小混亂 → 安穩
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：分享
- 陪伴角色：一顆會發光的小石頭
- 主導感官：視覺
- 睡前收束：顏色變淡、風變輕、念頭一個一個睡著
- 結尾句式：明日式  → 範式「雨芯，晚安。～還在那裡等明天，先安心睡吧。」
- 禁用元素：
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「閣樓上的舊鐘塔」
  - 不要主角慢慢從一個狀態變成另一個
  - 不要重複場景「圖書館深處的安靜角落」
  - 不要和前兩晚用同一種解決方式或結尾句式
