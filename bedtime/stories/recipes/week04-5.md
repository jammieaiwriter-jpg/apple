# 故事配方卡 — week04-5（合作）

- 切點（focus，固定）：完成後分享成果與功勞
- 故事形狀：製作型（一起動手做或修一個東西）
- 主角：一顆滾來滾去的小皮球
- 場景：結霜的玻璃窗內側
- 情緒曲線：擔心 → 被理解 → 安心
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：修補
- 陪伴角色：一隻打瞌睡的小貓
- 主導感官：味覺
- 睡前收束：動作變慢、被子變暖、世界變安靜
- 結尾句式：願式  → 範式「雨芯，晚安。願～，做一個～的夢。」
- 禁用元素：
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「夜晚的麵包店」
  - 不要循線索找一個遺失的東西或聲音
  - 不要重複場景「星星掉下來的草原」
  - 不要和前兩晚用同一種解決方式或結尾句式
