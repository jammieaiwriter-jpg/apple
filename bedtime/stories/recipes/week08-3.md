# 故事配方卡 — week08-3（道歉）

- 切點（focus，固定）：聽對方說感受，也尊重對方暫時不接受
- 故事形狀：相遇型（兩人各做各的後相遇、化解小誤會）
- 主角：一隻慢吞吞的小烏龜
- 場景：結霜的玻璃窗內側
- 情緒曲線：興奮 → 鼓起勇氣 → 安心
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：放下
- 陪伴角色：一個會回聲的空罐子
- 主導感官：嗅覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：物件呼應式  → 範式「雨芯，晚安。～（故事裡的物件）也～了，妳也～。」
- 禁用元素：
  - 不要主角慢慢從一個狀態變成另一個
  - 不要重複場景「閣樓上的舊鐘塔」
  - 不要循線索找一個遺失的東西或聲音
  - 不要重複場景「沙灘上的貝殼小屋」
  - 不要和前兩晚用同一種解決方式或結尾句式
