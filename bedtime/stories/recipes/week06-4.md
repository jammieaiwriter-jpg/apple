# 故事配方卡 — week06-4（嘗試）

- 切點（focus，固定）：遇到困難時願意請人幫忙
- 故事形狀：相遇型（兩人各做各的後相遇、化解小誤會）
- 主角：一片小小的雪花
- 場景：一格一格的棋盤花園
- 情緒曲線：孤單 → 小混亂 → 暖暖的
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：深呼吸
- 陪伴角色：會打噴嚏的麵粉罐
- 主導感官：視覺
- 睡前收束：動作變慢、被子變暖、世界變安靜
- 結尾句式：身體放鬆式  → 範式「雨芯，晚安。讓～慢慢～，～地睡。」
- 禁用元素：
  - 不要安靜守候一件事慢慢發生
  - 不要重複場景「雲上的小陽台」
  - 不要輕輕守護、照顧一個更小更弱的對象
  - 不要重複場景「圖書館深處的安靜角落」
  - 不要和前兩晚用同一種解決方式或結尾句式
