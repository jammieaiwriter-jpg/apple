# 故事配方卡 — week12-4（成長）

- 切點（focus，固定）：陪伴別人成長，但不替對方完成
- 故事形狀：相遇型（兩人各做各的後相遇、化解小誤會）
- 主角：一朵棉花雲
- 場景：睡前的小臥室
- 情緒曲線：孤單 → 小混亂 → 滿足
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：唱歌
- 陪伴角色：一盞不會說話的小夜燈
- 主導感官：味覺
- 睡前收束：香味變淡、燈光變柔、聲音變小
- 結尾句式：比喻式  → 範式「雨芯，晚安。今晚就像～，～。」
- 禁用元素：
  - 不要主角慢慢從一個狀態變成另一個
  - 不要重複場景「書頁之間的微小世界」
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「閣樓上的舊鐘塔」
  - 不要和前兩晚用同一種解決方式或結尾句式
