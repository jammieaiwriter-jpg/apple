# 故事配方卡 — week04-4（合作）

- 切點（focus，固定）：意見不同時一起找第三種方法
- 故事形狀：尋找型（循線索找一個遺失的東西或聲音）
- 主角：一隻迷你小火車
- 場景：星星掉下來的草原
- 情緒曲線：想念 → 被理解 → 安穩
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：觀察
- 陪伴角色：一個會回聲的空罐子
- 主導感官：觸覺
- 睡前收束：呼吸變慢、身體變暖、眼睛變重
- 結尾句式：比喻式  → 範式「雨芯，晚安。今晚就像～，～。」
- 禁用元素：
  - 不要兩人各做各的後相遇、化解小誤會
  - 不要重複場景「屋頂水塔旁的小天台」
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「夜晚的麵包店」
  - 不要和前兩晚用同一種解決方式或結尾句式
