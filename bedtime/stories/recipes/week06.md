# 故事配方卡 — week06（嘗試）

- 切點（focus，固定）：還沒有把握，也願意開始
- 故事形狀：變化型（主角慢慢從一個狀態變成另一個）
- 主角：一隻想念朋友的小狐狸
- 場景：結霜的玻璃窗內側
- 情緒曲線：興奮 → 驚喜 → 滿足
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：合作
- 陪伴角色：一隻安靜的螢火蟲
- 主導感官：味覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：物件呼應式  → 範式「雨芯，晚安。～（故事裡的物件）也～了，妳也～。」
- 禁用元素：
  - 不要出門遠行、一路過關
  - 不要重複場景「抽屜裡的玩具城」
  - 不要和前兩晚用同一種解決方式或結尾句式
