# 故事配方卡 — week10-2（選擇）

- 切點（focus，固定）：比較不同選擇可能帶來的結果
- 故事形狀：尋找型（循線索找一個遺失的東西或聲音）
- 主角：一滴小水珠
- 場景：月亮背面的小郵局
- 情緒曲線：興奮 → 慢慢明白 → 安穩
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：傾聽
- 陪伴角色：一條舊圍巾
- 主導感官：味覺
- 睡前收束：顏色變淡、風變輕、念頭一個一個睡著
- 結尾句式：物件呼應式  → 範式「雨芯，晚安。～（故事裡的物件）也～了，妳也～。」
- 禁用元素：
  - 不要一起動手做或修一個東西
  - 不要重複場景「書頁之間的微小世界」
  - 不要和前兩晚用同一種解決方式或結尾句式
