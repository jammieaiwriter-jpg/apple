# 故事配方卡 — week12-5（成長）

- 切點（focus，固定）：接受自己仍在學習，帶著勇氣繼續前進
- 故事形狀：守護型（輕輕守護、照顧一個更小更弱的對象）
- 主角：一隻小章魚
- 場景：沙灘上的貝殼小屋
- 情緒曲線：緊張 → 猶豫 → 安心
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：放下
- 陪伴角色：一隻打瞌睡的小貓
- 主導感官：嗅覺
- 睡前收束：呼吸變慢、身體變暖、眼睛變重
- 結尾句式：願式  → 範式「雨芯，晚安。願～，做一個～的夢。」
- 禁用元素：
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「閣樓上的舊鐘塔」
  - 不要兩人各做各的後相遇、化解小誤會
  - 不要重複場景「睡前的小臥室」
  - 不要和前兩晚用同一種解決方式或結尾句式
