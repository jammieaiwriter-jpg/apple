# 故事配方卡 — week08（道歉）

- 切點（focus，固定）：發現自己的行動影響了別人
- 故事形狀：變化型（主角慢慢從一個狀態變成另一個）
- 主角：一個小小的鬧鐘精靈
- 場景：閣樓上的舊鐘塔
- 情緒曲線：孤單 → 猶豫 → 放鬆
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：修補
- 陪伴角色：窗外的月亮
- 主導感官：觸覺
- 睡前收束：動作變慢、被子變暖、世界變安靜
- 結尾句式：身體放鬆式  → 範式「雨芯，晚安。讓～慢慢～，～地睡。」
- 禁用元素：
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「星星掉下來的草原」
  - 不要和前兩晚用同一種解決方式或結尾句式
