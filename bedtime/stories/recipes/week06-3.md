# 故事配方卡 — week06-3（嘗試）

- 切點（focus，固定）：一種方法不行，就換一種試試看
- 故事形狀：守護型（輕輕守護、照顧一個更小更弱的對象）
- 主角：一把會唱歌的小雨傘
- 場景：圖書館深處的安靜角落
- 情緒曲線：想念 → 發現 → 放鬆
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：擁抱
- 陪伴角色：一把溫柔的小掃帚
- 主導感官：嗅覺
- 睡前收束：顏色變淡、風變輕、念頭一個一個睡著
- 結尾句式：比喻式  → 範式「雨芯，晚安。今晚就像～，～。」
- 禁用元素：
  - 不要主角慢慢從一個狀態變成另一個
  - 不要重複場景「結霜的玻璃窗內側」
  - 不要安靜守候一件事慢慢發生
  - 不要重複場景「雲上的小陽台」
  - 不要和前兩晚用同一種解決方式或結尾句式
