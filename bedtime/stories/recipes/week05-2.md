# 故事配方卡 — week05-2（傾聽）

- 切點（focus，固定）：注意表情、動作與沒有說出口的訊息
- 故事形狀：變化型（主角慢慢從一個狀態變成另一個）
- 主角：一隻穿雨鞋的小青蛙
- 場景：圖書館深處的安靜角落
- 情緒曲線：孤單 → 發現 → 安穩
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：等待
- 陪伴角色：一條舊圍巾
- 主導感官：嗅覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：願式  → 範式「雨芯，晚安。願～，做一個～的夢。」
- 禁用元素：
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「閣樓上的舊鐘塔」
  - 不要和前兩晚用同一種解決方式或結尾句式
