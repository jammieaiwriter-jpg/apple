# 故事配方卡 — week11-2（想像）

- 切點（focus，固定）：勇敢提出「如果……會怎樣？」
- 故事形狀：旅程型（出門遠行、一路過關）
- 主角：一塊害羞的小餅乾
- 場景：抽屜裡的玩具城
- 情緒曲線：害羞 → 小混亂 → 暖暖的
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：分享
- 陪伴角色：一條舊圍巾
- 主導感官：視覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：明日式  → 範式「雨芯，晚安。～還在那裡等明天，先安心睡吧。」
- 禁用元素：
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「森林邊的小木屋」
  - 不要和前兩晚用同一種解決方式或結尾句式
