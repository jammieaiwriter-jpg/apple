# 故事配方卡 — week08-5（道歉）

- 切點（focus，固定）：下次遇到相同情況時做出改變
- 故事形狀：守護型（輕輕守護、照顧一個更小更弱的對象）
- 主角：一隻想念朋友的小狐狸
- 場景：圖書館深處的安靜角落
- 情緒曲線：好奇 → 被理解 → 放鬆
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：擁抱
- 陪伴角色：一顆會發光的小石頭
- 主導感官：聽覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：願式  → 範式「雨芯，晚安。願～，做一個～的夢。」
- 禁用元素：
  - 不要兩人各做各的後相遇、化解小誤會
  - 不要重複場景「結霜的玻璃窗內側」
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「月亮背面的小郵局」
  - 不要和前兩晚用同一種解決方式或結尾句式
