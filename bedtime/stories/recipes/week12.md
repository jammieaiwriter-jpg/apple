# 故事配方卡 — week12（成長）

- 切點（focus，固定）：回頭看見自己已經做到的改變
- 故事形狀：尋找型（循線索找一個遺失的東西或聲音）
- 主角：一片飄下來的落葉
- 場景：夜晚的麵包店
- 情緒曲線：想念 → 鼓起勇氣 → 暖暖的
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：修補
- 陪伴角色：一個會回聲的空罐子
- 主導感官：視覺
- 睡前收束：香味變淡、燈光變柔、聲音變小
- 結尾句式：身體放鬆式  → 範式「雨芯，晚安。讓～慢慢～，～地睡。」
- 禁用元素：
  - 不要主角慢慢從一個狀態變成另一個
  - 不要重複場景「雨後的彩虹橋下」
  - 不要和前兩晚用同一種解決方式或結尾句式
