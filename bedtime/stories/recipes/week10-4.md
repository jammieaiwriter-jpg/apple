# 故事配方卡 — week10-4（選擇）

- 切點（focus，固定）：做出決定，也接受沒有選到的部分
- 故事形狀：相遇型（兩人各做各的後相遇、化解小誤會）
- 主角：一隻迷你小火車
- 場景：下雪的屋頂
- 情緒曲線：害羞 → 猶豫 → 放鬆
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：放下
- 陪伴角色：一顆會發光的小石頭
- 主導感官：聽覺
- 睡前收束：呼吸變慢、身體變暖、眼睛變重
- 結尾句式：身體放鬆式  → 範式「雨芯，晚安。讓～慢慢～，～地睡。」
- 禁用元素：
  - 不要循線索找一個遺失的東西或聲音
  - 不要重複場景「月亮背面的小郵局」
  - 不要到別人的地方作客、靠對話推進
  - 不要重複場景「池塘邊的荷葉舞台」
  - 不要和前兩晚用同一種解決方式或結尾句式
