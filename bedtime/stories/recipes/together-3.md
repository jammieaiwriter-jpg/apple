# 故事配方卡 — together-3（一起玩）

- 切點（focus，固定）：用邀請代替命令，朋友才會留下來
- 故事形狀：製作型（一起動手做或修一個東西）
- 主角：一支愛發號施令的小指揮棒
- 場景：公園角落的沙坑
- 情緒曲線：想掌控 → 夥伴一個個安靜走開的小失落 → 安心
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：分享
- 陪伴角色：幾把一被命令就安靜走開的小鏟子
- 主導感官：聽覺
- 睡前收束：聲音變小、燈光變柔、動作變慢
- 結尾句式：明日式  → 範式「雨芯，晚安。～還在那裡等明天，先安心睡吧。」
- 禁用元素：
  - 不要主角被責罵或處罰才改變（要他自己發現命令把朋友趕走、邀請把朋友找回來）
  - 不要主角慢慢從一個狀態變成另一個（不要變化型）
  - 不要重複場景「開滿小花的長下坡小路」
  - 不要和前兩晚用同一種解決方式或結尾句式
  - 對照「命令的語氣」與「邀請的語氣」帶來的不同，但不要貼壞標籤、不訓話
