# 故事配方卡 — week09-5（耐心）

- 切點（focus，固定）：分辨哪些事情能加快，哪些需要時間
- 故事形狀：等待型（安靜守候一件事慢慢發生）
- 主角：一隻想念朋友的小狐狸
- 場景：森林邊的小木屋
- 情緒曲線：好奇 → 發現 → 放鬆
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：唱歌
- 陪伴角色：一隻不會說話的紙鶴
- 主導感官：味覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：身體放鬆式  → 範式「雨芯，晚安。讓～慢慢～，～地睡。」
- 禁用元素：
  - 不要兩人各做各的後相遇、化解小誤會
  - 不要重複場景「海邊的小燈塔」
  - 不要出門遠行、一路過關
  - 不要重複場景「會發光的蘑菇山洞」
  - 不要和前兩晚用同一種解決方式或結尾句式
