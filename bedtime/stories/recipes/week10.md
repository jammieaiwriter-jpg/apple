# 故事配方卡 — week10（選擇）

- 切點（focus，固定）：先停下來，想清楚真正需要什麼
- 故事形狀：製作型（一起動手做或修一個東西）
- 主角：一隻夜裡才醒的小貓頭鷹
- 場景：書頁之間的微小世界
- 情緒曲線：興奮 → 被理解 → 安心
- 核心事件：（依切點自填一句：今晚發生什麼小事）
- 解決方式：合作
- 陪伴角色：一個會回聲的空罐子
- 主導感官：觸覺
- 睡前收束：影子變短、月光變亮、心跳變慢
- 結尾句式：願式  → 範式「雨芯，晚安。願～，做一個～的夢。」
- 禁用元素：
  - 不要安靜守候一件事慢慢發生
  - 不要重複場景「森林邊的小木屋」
  - 不要和前兩晚用同一種解決方式或結尾句式
